INSTALLATION

Diese 3 Programme einmal installieren — dann laufen alle 26 Mods

1
Script Hook V
Das Basis-Programm für alle GTA V Mods. Ohne Script Hook V läuft kein einziger Mod. Wird nach jedem GTA Update von Alexander Blade aktualisiert.
📁 GTA V Hauptordner (wo GTA5.exe liegt)
Dateien: ScriptHookV.dll + dinput8.dll
🔗 dev-c.com/gtav/scripthookv → ZIP runterladen
2
SHVDN Nightly
ScriptHookVDotNet – macht dass GTA V die .cs Mod-Dateien laden kann. Immer die NEUESTE Nightly-Version nehmen!
📁 GTA V Hauptordner (wo GTA5.exe liegt)
Dateien: ScriptHookVDotNet.asi + ScriptHookVDotNet2.dll + ScriptHookVDotNet3.dll
🔗 GitHub → neuestes ZIP → 3 Dateien rein
3
NativeUI
Zeichnet die Menüs im Spiel. Ohne NativeUI.dll erscheint kein Menü wenn du eine Taste drückst. Nur eine Datei nötig.
📁 scripts Ordner → GTA V Hauptordner\scripts\
Datei: NativeUI.dll
🔗 GitHub → Release.zip → NativeUI.dll in scripts
4
Mod installieren
Wenn 1-3 fertig: Mod herunterladen, ZIP entpacken, .cs Datei in den scripts Ordner. GTA V starten – Mod lädt automatisch. Fertig!
📁 scripts Ordner → GTA V Hauptordner\scripts\
Datei: ModName.cs (aus der ZIP)

BATTLEYE

Muss ich das deaktivieren?

🛡️
Ja – BattlEye muss deaktiviert sein

BattlEye ist Rockstars Anti-Cheat und blockiert Script Hook V automatisch wenn es aktiv ist. Im Story Mode hat BattlEye keine andere Funktion – es schützt nur GTA Online. Das Deaktivieren ist vollständig sicher solange du nur im Story Mode spielst.

Wichtig: Niemals GTA Online spielen wenn BattlEye deaktiviert ist – Rockstar bannt Accounts dafür.

Epic Games Launcher → Bibliothek → GTA V → drei Punkte (...) → Eigenschaften
Rockstar Games Launcher → GTA V Einstellungen → Anti-Cheat deaktivieren
BattlEye Haken entfernen / deaktivieren
GTA V neu starten → Script Hook V und alle Mods funktionieren jetzt
